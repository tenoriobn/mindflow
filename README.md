# MindFlow

Landing page moderna desenvolvida para o projeto **MindFlow**, uma plataforma focada em clareza mental, produtividade consciente e equilíbrio emocional.

## ✨ Preview

![Mindflow Preview](./public/preview.jpg)

🔗 Deploy:  
https://mindflowdigital.vercel.app/

O projeto foi desenvolvido com foco em:

- experiência visual premium;
- motion design suave;
- responsividade;
- acessibilidade;
- performance;
- SEO moderno.

---

## 🚀 Tecnologias

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- GSAP
- Video.js
- SVGR
- ESLint
- Prettier

---

## 🧠 Recursos

- Landing page totalmente responsiva
- Animações com GSAP + ScrollTrigger
- Video player customizado
- Accordion interativo para FAQ
- SEO otimizado com Metadata API
- Estrutura modular e escalável
- Motion design focado em UX

## ⚙️ Executando o Projeto

### Clone o repositório

```bash
git clone https://github.com/seu-usuario/mindflow.git
```

### Instale as dependências

```bash
npm install
```

### Execute o projeto

```bash
npm run dev
```

A aplicação estará disponível em:

```bash
http://localhost:3000
```
